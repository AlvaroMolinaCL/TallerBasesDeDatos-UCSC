# Taller de Bases de Datos
Este repositorio contiene material de estudio, códigos y ejercicios resueltos en el curso de Taller de Bases de Datos (IN1078C) impartido en la UCSC (específicamente en el semestre 2024-1).

**Importante**: El sistema gestor de base de datos (SGBD) utilizado durante dicho curso fue PostgreSQL y el lenguaje procedural usado fue el mismo que provee el SGBD (es decir, PL/pgSQL).
